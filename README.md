## Getting Started

First, install node_module:
```bash
npm install
# or
yarn
# or
bun
```
to build
```bash
npm run build
# or
yarn build
# or
bun build
```
to preview
```bash
npm run preview
# or
yarn preview
# or
bun preview
```
run the development server:

```bash
npm run dev
# or
yarn dev
# or
bun dev
```
Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
